# 7-Minute Demo Script

## 0:00 — Positioning

"This is an AI Analyst command center. It does not just summarize data. It runs a governed analytics pipeline: data quality, semantic setup, anomaly detection, root cause, forecast, guardian review, and evidence-based recommendations."

## 0:45 — Load demo data

Use `Fintech onboarding demo`. Explain that it simulates a secured-card onboarding journey with signup, verification, payment, VKYC, card issue, and activation stages.

## 1:30 — Validate schema

Show auto-detected date column and KPI column. Mention that the user can override them before analysis.

## 2:00 — Run governed analysis

Click **Run governed analysis**. Point to the live agent console:

- EDA
- Hypothesis
- Trend
- Anomaly
- Root cause
- Funnel
- Cohort
- Forecast
- Debate
- Guardian
- Insight

## 3:00 — Executive brief

Show the final brief. Explain that it is generated from structured agent findings and passed through the security/output boundary.

## 4:00 — Deep dive

Show KPI trend, anomaly count, driver attribution, funnel, cohort, and forecast sections.

## 5:30 — Ask follow-up

Ask: "What drove the drop?" or "Which agents ran?" Explain that chat is grounded in the completed analysis context.

## 6:30 — Export

Download Markdown or JSON. Mention that the JSON includes run metadata and can be connected to audit/replay.

## 7:00 — Close

"The value is not only the UI. The value is the governed architecture behind it: one canonical pipeline, trust controls, agent evidence, and a product workflow that makes it usable by business teams."
